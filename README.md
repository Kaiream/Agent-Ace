# Agent Ace

This project is a C# program developed by me as part of my university study program. The project is open-source and is playable on any PC that can run .exe and has .NET 9.0 installed. No AI was used in making this project.

## Dependencies

.NET 9.0 downloadable at: https://dotnet.microsoft.com/en-us/download/dotnet/9.0

## Documentation
**Project topic: The object avoids falling objects**

Initial idea: The game is going to be about a guy named Ace. He works for an underground mafia and has to doge police vehicles.
Kind of like flappy bird or space invaders.

Features:
- In-game shop
- Built-in graphics using System.Drawing
- Progressive difficulty
- Save & Load functionality
